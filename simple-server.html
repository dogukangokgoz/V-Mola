Mola Takip Sistemi

Çalışanların mola sürelerini takip etmek için geliştirilmiş modern web uygulaması

⚠️ Node.js ve PostgreSQL kurulumu gerekiyor

🚀 Kurulum Adımları

1. Node.js kurun: https://nodejs.org
2. PostgreSQL kurun: brew install postgresql
3. Bağımlılıkları yükleyin: npm install
4. Veritabanını kurun: createdb mola_takip
5. Uygulamayı başlatın: npm run dev-full

📋 Detaylı Kurulum 📚 Dokümantasyon

📋 Detaylı Kurulum Talimatları

1. Node.js Kurulumu

https://nodejs.org adresinden LTS versiyonunu indirin ve kurun.

2. PostgreSQL Kurulumu

Terminal'de şu komutları çalıştırın:

brew install postgresql
brew services start postgresql

3. Proje Kurulumu

Terminal'de proje klasöründe:

npm install
cd client && npm install && cd ..

4. Veritabanı Kurulumu

createdb mola_takip
psql -d mola_takip -f database/schema.sql

5. Çevre Değişkenleri

cp env.example .env

Sonra .env dosyasını düzenleyin.

6. Uygulamayı Başlatın

npm run dev-full

✅ Başarılı olursa http://localhost:3000 adresinde uygulamayı görebilirsiniz!
